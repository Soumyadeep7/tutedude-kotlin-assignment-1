# Kotlin Basics Practice – Module 3 Assignment

This repository contains solutions to the **Module 3 Kotlin Basics Practice Assignment**.
It covers fundamental concepts of Kotlin including conditionals, functions, arrays, and lists.

---




---

## ✅ Section A: Basic Practice

1. Check whether a number is even or odd
2. Find the largest of three numbers
3. Print multiplication table
4. Count vowels in a string
5. Reverse a string

---

## ✅ Section B: Conditionals & Functions

6. Grade calculator using `when`
7. Simple calculator using functions
8. Palindrome number check
9. Menu-driven calculator
10. Factorial function

---

## ✅ Section C: Arrays & Lists

11. Sum of array elements
12. Maximum element in a list
13. Remove duplicates from a list
14. Frequency count of elements
15. Second largest element

## ▶️ How to Run

You can run the programs using:

* Kotlin CLI

Each file contains a `main()` function and can be run independently.
